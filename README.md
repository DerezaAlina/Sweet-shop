# 🍰 Sweet-shop

Sweet-shop is project for ordering sweet. Homepage on the link <http://localhost:3000/main>

## ⬇️ Installation

1. Clone the repo:
```bash
gh repo clone DerezaAlina/Sweet-shop
```
2. Install NPM packages:
```
npm install
```

## 🚀 Usage
npm start

## 📓 Run tests
npm test
